## Trabalho interdiciplinar ITB

Olá pessoal,

Gostaria de compartilhar um projeto que fiz no último trimestre da escola. Foi um trabalho onde tive que criar um site para adicionar conteúdos de várias disciplinas técnicas. Cada professor pediu um tipo de conteúdo específico e eu tive que adicionar no site usando as linguagens HTML, CSS e JavaScript.

Para começar, criei a estrutura básica do site usando HTML. Adicionei uma barra de navegação no topo da página para permitir que os usuários navegassem facilmente pelas diferentes seções do site. Usei também HTML para criar caixas de texto e botões para interagir com o usuário.

Em seguida, apliquei CSS para formatar o site e torná-lo mais atraente visualmente. Adicionei cores, fontes e imagens para destacar os diferentes tipos de conteúdo e tornar a experiência do usuário mais agradável.

Finalmente, usei JavaScript para adicionar algumas funcionalidades extras ao site. Adicionei uma animação de rolagem suave quando o usuário navega pela página, e também adicionei algumas interações para permitir que os usuários filtrarem o conteúdo de acordo com suas preferências.

Foi um projeto muito desafiador, mas aprendi muito sobre as linguagens HTML, CSS e JavaScript ao longo do processo. Aprendi a criar estruturas de página complexas, a formatar elementos de página e a adicionar interações e animações. Acredito que essas habilidades serão muito úteis à medida que eu avanço na minha carreira em programação.


### ✔️ Técnicas e tecnologias utilizadas

- ``HTML``
- ``CSS``
- ``JavaScript``


Obrigado por ler sobre o meu projeto e espero que isso inspire outras pessoas a experimentar a programação.
